正文:
这是一个更新频繁的软件。
很多书上的指令在MongoDB v3.6用不了。
查看系统服务 run + services.msc（检查mongodb服务是否启动）

简言之
1、下载解压安装
2、exe所在文件夹添加到系统环境变量。
3、创建数据存储文件夹 data\db，日志文件夹 log（需要手动创建）
下面都是两个短横，没显示完全
run cmd as admin
mongod –directoryperdb –dbpath C:\mongodb\data\db –logpath C:\mongodb\log\mongo.log –logappend –install
4、net start MongoDB启动服务
5、net stop MongoDB关闭服务
