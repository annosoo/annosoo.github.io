要点整理

发表于 2019-08-17 分类于 Python ， 《python编程导论》

前言

计算机做两件事。执行计算和保存计算。

对象

对象和操作符可以组成表达式。

对象有两种类型。
标量对象:整数、实数、布尔值、None
非标量对象:字符串。有内部结构

变量

python中变量只是名称
赋值语句将左边的名称与右边的对象关联。
对象可以和多个名称关联，也可以不关联。

变量可以包含大小写字母、数字、下划线。
不能以数字开头，不能用保留字。
变量大小写敏感。

python支持多重赋值。
赋值语句先对右边的每个表达式求值，然后和左边的变量名绑定。

操作符

操作符+ *被称为重载
‘a’ + ‘a’ 连接
n * ‘a’ 重复

操作符优先级。

字符串

字符串是序列类型之一。
len函数求字符串长度。
索引提取单个字符。
分片操作提取任意长度子串。s[start : end]表示从s中提取start开始，end-1结束的子串。

input参数为字符串。shell中作为提示信息，用户输入成为函数的返回值。

当print语句有多个参数时，会在每个参数对应的值之间加上空格。print(‘a’, ‘a’)
字符串连接后生成一个没有多余空格的字符串作为唯一参数。print(‘a’ + ‘a’)

类型转换

字符编码

# -- coding utf-8 --

迭代

需要程序多次做同一件事。

小技巧

ctrl + c 强制结束

用print语句测试

三引号之间的文本称为文档字符串，提供函数的规范。规范包括:假设和保证。假设是对实参的限制，保证是函数应当实现的功能。

help(abs) 查找帮助

rang()函数

range(start, stop, step)
只有两个参数。
只有一个参数。
range(0, 3)和range(3)都生成序列0,1,2

for循环

for循环配合in操作符也可以遍历字符串中的字符

break语句提前终止循环

几种算法比较

穷举法和二分法

比较两个浮点数，最好的做法是看是否足够接近，而不是是否相等。
float类型的值以浮点数的形式保存。采用二进制表示法。

牛顿-拉弗森法。也就逐次逼近。
如果guess是多项式p的根的近似值，那guess-p(guess)/p’(guess)就是一个更好的近似值。作为下一个猜测。

函数

函数。方便扩展和重用代码。
return语句只能用在函数体中。
函数被调用时，形参被绑定到实参。

位置参数。关键字参数。
关键字参数经常和默认参数值配合使用。

函数是一个黑盒。
分解和抽象。
分解▶结构化。
抽象▶隐藏细节。

递归

自然数的阶乘函数是简单的递归定义。

斐波那契数列也是递归定义。

基本情形和递归情形。

回文字符串。一个字符串顺读和倒读一样。

全局变量global

模块

模块是包含Python定义和语句的.py文件。

import M
from M import *

文件

Python通过文件句柄处理文件
fh = open(fn, ‘w’) fn是文件名的字符串
fh = open(fn, ‘r’)
fh = open(fn, ‘a’)
fh.read()
fh.readline()
fh.readlines()
fh.write(s)
fh.writeLines(S)
fh.close()

结构化类型

另外4种结构化类型。非变量对象。
tuple、list、range、dict

元组是一些元素的不可变有序序列。
包含1的单元素元组。(1, )
小括号用来分组表达式。

元组可以进行连接、索引、切片、重复、len()、in、not in、for遍历操作

多重赋值。提取序列中单个元素。

range最常用在for循环

列表、字典可变。
元组、字符串、范围不可变。

L.count(e)返回e在L中出现的次数
L.index(e)
L.insert(i, e)
L.remove(e)
L.pop(i)
L.sort()升序排列L中的元素
L.reverse()翻转L中的元素顺序
L.extend(L1)追加L1中的元素
L.append(e)追加对象e。保持原来的结构。

修改一个正在遍历的列表。
切片克隆。for e in L[:]
或者list(L)返回列表L的副本。
或者导入标准库模块。copy.deepcopy(L)

列表推导式

将某种操作应用到序列中的每个元素上。
创建一个新的列表。
L = [x**2 for x in range(1, 7)]
for 从句前是操作。for从句生成序列。for从句后可再加限制条件。

函数对象

内置高阶函数map和for循环配合使用。

map函数第一个参数是函数。第二个参数是有序集合。作为第一个参数的实参。
循环每次迭代返回一个值。

匿名函数lambda

lambda x, y : x ** y
返回一个函数。
常和map函数配合使用。

元组和列表比较

元组相比列表，可以作为字典的键。

字符串操作

s.count(s1)返回s1在s中出现的次数
s.find(s1)
s.rfind(s1)
s.index(s1)
s.rindex(s1)
s.lower()大写字母转换成小写
s.replace(old, new)
s.rstrip()去掉s末尾的空白字符
s.split(d)用d作为分隔符拆分s。如省略，用任意空白字符拆分。

空白字符

空白字符包括:空格、制表符、换行符、回车、分页符

字典

字典是键/值对的集合。
用键对值进行索引。

for遍历字典。分给迭代变量的值是字典的键

字典键必须是可散列类型的对象。
不可变类型都是可散列。如元组。如用(flightNumber, day)表示航空公司的航班。作为字典的键。
可变类型都是不可散列。

字典操作

len(d)
d.keys()返回所有键的视图
d.values()返回所有值的视图
k in d
d[k]用键索引值
d.get(k, v)
d[k] = v
del d[k]
for k in d:遍历d中的键

测试和调试

黑盒测试。基于规范探索路径
白盒测试。基于代码探索路

测试包括单元测试和集成测试。

调试是修复软件缺陷的过程。

异常处理

异常处理try-except-else-finally代码块

except后可接异常元组

抛出异常raise语句
raise exceptionName(arguments)
参数用来描述引发异常的原因

except exceptionName as msg:
print(msg)
将msg绑定到抛出exceptionName时使用的参数。用于打印错误信息。

断言assert语句
assert Boolean expression
assert Boolean expression, argument
对布尔表达式求值。True向下执行。False抛出AssersionError异常。

面向对象的设计思想

抽象出类，再根据类创建具体的实例

对象包含数据和操作数据的函数

__init__方式用于初始化。

print命令会自动调用与待输出对象关联的str方法。

类的内部定义访问数据的函数称为方法

方法可以直接在实例上调用

判断一个变量是否是某个类型或者该类型的父继承链
isinstance()

获取对象的所有属性dir()

len函数内部实际是调用该对象的len方法
len(str) 等价于 str.len()

类变量

类名.类变量名
所有实例共享的变量
存在于类定义中，不在方法内。

实例变量

实例名.实例变量名
各个实例本身独有
存在于方法中

hasattr()判断属性是否存在
getattr()获取属性
setattr()设置属性

类和实例的关系

实例创建后就可以自由绑定属性。动态绑定。

类是抽象，实例是具体

类的属性，所有实例均可调用。实例名作为参数传入形参self

子类继承父类属性，父类又叫基类、超类

python允许多重继承。一个子类可以继承多个父类属性
Mixin混入。组合不同类的功能

__slot__变量限制当前类能添加的属性

@property限制参数
